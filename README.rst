`Mopidy <http://www.mopidy.com/>`_ extension to access the Austrian radio station
`OE1 <http://oe1.orf.at/>`_. It provides access to the live stream and the 7-day-archive.

Installation
============

Install by running::

    sudo pip install Mopidy-OE1
